# Gambar
A cli tool to generate ASCII Art from image.

## Usage
```
gambar /path/to/image

# Or save it to a file
gambar /path/to/image > image.txt
cat image.txt
```

## Installation
```
# MacOS
brew install gd

# Linux
sudo apt-get libgd-dev

# Install dependencies
git clone https://github.com/sendyhalim/gambar
cd gambar
make install
```
